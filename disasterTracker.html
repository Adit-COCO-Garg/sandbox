Why?

The aim of this project is two-fold: understand how to fully take advantage of libraries and APIs; and provide valuable disaster/ crisis related data instantly.

Disaster Tracker provides data visualization of recent disasters and current coronavirus updates.

What?

Disaster tracker gathers data from 3 different APIs and 5 different endpoints to map out and visualize disaster locations on a map and display any relevant information pretaining to the disaster

How?

Mapbox GL JS API is used to render a map upon which disaster data from Reliefweb and COVID-19 data from about-corona APIs is mapped out. When a marker belonging to either is selected Reliefweb's disaster endpoint and NEWS API respectively is used to query all relevant news and information about the selected marker/ region.
